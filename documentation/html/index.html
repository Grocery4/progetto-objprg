Requisiti

Per il corretto funzionamento del progetto è necessario includere i file .jar inerenti la creazione e gestione dei grafici. In particolare, i file necessari sono:

  • jcommon-1.0.23.jar
  • jfreechart-1.0.19.jar
  • jfreechart-1.0.19-swt.jar

Reperibili sul sito ufficiale della libreria: jfree.org

Esecuzione del programma

Per eseguire il programma è necessario prima compilare i file .java e successivamente eseguire il file principale contenente il metodo main. All'interno della directory del progetto sono presenti due script .bash che facilitano le due operazioni citate precedentemente.

compile.bash:

# -d: specifica dove piazzare i file .class generati dalla compilazione
# -cp specifica dove trovare le dipendenze del programma (.jar)
# .. : specifica quali sono i file target della compilazione
javac -d ./bin -cp ./lib/*:. src/App.java src/controller/Controller.java src/view/View.java src/view/components/*.java src/view/errors/*.java src/model/*.java src/model/esami/*.java

execute.bash:

# -cp: specfica dove trovare i file generati e le loro dipendenze.
# : è il separatore tra le varie directory
java -cp bin/:./lib/*:. App
